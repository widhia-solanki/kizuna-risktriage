# Kizuna RiskTriage

## Calibrated Uncertainty Quantification for Supply Chain Risk Triage

**Team Kizuna** | AI for Public Good: Sustainable & Resilient Supply Chains Hackathon

---

### Problem
Point forecasts hide risk. Most "confidence intervals" are uncalibrated — a stated 95% interval often covers the truth only ~70% during volatile periods. Managers are given false confidence exactly when inventory decisions are most expensive.

### Solution
**Kizuna RiskTriage** is a calibrated, uncertainty-aware demand forecasting and risk-triage system that:
1. Produces probabilistic forecasts (LightGBM + Quantile Regression)
2. Calibrates intervals using Conformalized Quantile Regression (CQR) + Adaptive Conformal Inference (ACI)
3. Translates uncertainty into actionable risk tiers (Low / Medium / High)
4. Demonstrates business value via newsvendor inventory simulation
5. Audits its own honesty under volatility (conditional coverage analysis)

### Dataset
M5 Forecasting Competition (Walmart) — 30,490 products, 5.4 years daily sales, with built-in volatility regimes (SNAP, holidays, promotions).

## Methods Compared

| Method | Adaptive? | Coverage Guarantee? | Handles Drift? |
|--------|-----------|-------------------|----------------|
| Quantile Regression | Yes | No | No |
| Split Conformal | No (constant width) | Yes (marginal) | No |
| **CQR (ours)** | **Yes** | **Yes (marginal)** | No |
| **CQR + ACI (ours)** | **Yes** | **Yes (long-run)** | **Yes** |

## Key Results (generated by pipeline)
- Calibration curves showing CQR+ACI achieves near-target coverage
- Conditional coverage audit: naive methods under-cover during volatility
- Risk tier validation: High-tier SKUs have higher realized stockout rates
- Inventory simulation: calibrated system hits service target at lower cost
